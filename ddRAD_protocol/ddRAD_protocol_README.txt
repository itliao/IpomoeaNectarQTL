File for a step-by-step protocol for preparing double-digest restriction assisted DNA (ddRAD) libraries for sequencing. 
This protocol is a combination of the following two references: 

Peterson et al., 2012 (https://journals.plos.org/plosone/article?id=10.1371/journal.pone.0037135)
Ostevik, 2016 (https://open.library.ubc.ca/cIRcle/collections/ubctheses/24/items/1.0228559)

Protocol: ddRAD_Protocol_180916.docx
Other Notes (referred in protocol): JR_Ligation_notes_for_Kate_and_Irene.docx
